# 구도 패턴과 독창성 규칙

## 기본 구조 유형

구조는 하나만 고른다. 너무 많이 섞지 않는다.

### Workflow 흐름

적합한 경우: 입력 -> 처리 -> 출력, 콘텐츠 생산, AI 워크플로, 자동화 체인.

그리는 법: 왼쪽에 입력, 가운데에 小黑 또는 이상한 처리 기계, 오른쪽에 출력을 둔다. 주황색 화살표로 주 흐름을 표현한다.

### 시스템 일부

적합한 경우: 정보 출처, 필터, 데이터베이스, 렌더러, agent 시스템의 일부.

그리는 법: 핵심 모듈 3-5개만 그리고, 小黑가 그중 하나의 핵심 동작에 참여한다.

### 전후 비교

적합한 경우: 혼란/질서, 수동/자동, 흩어짐/모임, 불안/안정.

그리는 법: 왼쪽은 혼란, 오른쪽은 안정, 가운데는 주황색 화살표. 캐릭터는 조금 과장해도 된다.

### 역할 상태

적합한 경우: 사용자 고통점, 창작자 상태, 도구 과다, 정보 불안, 막힘에서 작동으로 넘어가는 변화.

그리는 법: 2-4개의 작은 상태를 그리고, 각 상태에는 짧은 한국어 라벨 하나만 둔다.

### 개념 은유

적합한 경우: 콘텐츠 공장, 정보 창고, 머릿속 블랙박스, 워크플로 기계, 자동 일지.

그리는 법: 큰 이상한 물건이나 기계 하나, 적은 입력, 출력 하나. 기억에 남는 지점이 있어야 한다.

### 방법 분층

적합한 경우: 방법론 프레임워크, 시스템 계층, 능력 스택, 콘텐츠 시스템 계층.

그리는 법: 층층이 쌓인 상자를 그리되 공식 피라미드처럼 만들지 않는다. 小黑는 옆에서 벽돌을 나르거나 쌓는다.

### 지도 경로

적합한 경우: 아이디어에서 출시까지, 사용자 경로, 콘텐츠 이어받기 경로, 학습 경로.

그리는 법: 휘어진 길 하나와 적은 노드를 둔다. 小黑는 줄을 끌거나 길을 걷는다.

### 짧은 만화 컷

적합한 경우: 실패에서 성공으로, 실제 과정, 짧은 풍자, 사용 전후 변화.

그리는 법: 2-4개의 작은 장면으로 나누고, 각 칸은 하나의 동작만 표현한다.

## 독창적 은유 생성법

매번 현재 글에서 은유를 다시 발명한다. 기존 그림을 그대로 가져오지 않는다.

### 세 단계

1. 추상 개념을 물리적 동작으로 바꾼다: 끼임, 새어 나감, 무거워짐, 분류, 침전, 발효, 문 열기, 접기, 포장 풀기, 되돌아 흐르기.
2. 시스템 구조를 낮은 기술의 물건으로 바꾼다: 고장 난 기계, 종이상자, 서랍, 수도관, 우체통, 이상한 계기판, 저울, 우물, 사다리, 이상한 작업대.
3. 小黑가 동작을 맡게 한다: 옆에 서 있지 않고, 기계에 끼거나, 잘못된 줄을 당기거나, 문을 지키거나, 운반하거나, 수리하거나, 무게를 재거나, 사다리를 붙잡거나, 기록하거나, 무언가를 이상한 장치에 밀어 넣는다.

### 사용할 수 있는 물건 풀

- 종이상자, 서랍, 낡은 기계, 깔때기, 저울, 우체통, 문, 우물, 사다리, 수도관, 실타래, 수문, 턴테이블, 블랙박스, 구멍 뚫는 도구, 면 압착기, 빨랫줄, 이상한 작업대.
- 한 번에 1-2개만 고르고 가득 쌓지 않는다.

### 小黑 동작 풀

- 당기기, 짊어지기, 밀어 넣기, 건지기, 누르기, 무게 재기, 꿰매기, 자르기, 비틀기, 지키기, 밀기, 받기, 뜯기, 표시하기, 회수하기.
- 동작은 핵심 의미를 위해 존재해야 한다. 이상함만을 위한 이상함은 피한다.

## 반복 복제 방지 규칙

기본적으로 `assets/examples/`를 열거나 복제하지 않는다. 이 이미지는 선 밀도, 여백, 색 절제, 小黑의 분위기 같은 스타일 보정에만 사용한다.

사용자가 “이 그림처럼 / 이 구도 복각 / 이 사례로 수정”이라고 명시하지 않는 한 다음 기존 구도를 직접 재사용하지 않는다.

- 컨베이어의 두 단절점
- 小黑가 콘텐츠 기계 안에서 판단 레버를 당기는 구도
- 小黑가 깔때기가 되어 트래픽/신뢰/전환을 분류하는 구도
- 小黑가 소재 물고기를 자르는 구도
- 小黑가 이어받는 경로를 끄는 구도
- 小黑가 세 층의 정보원을 당기는 구도
- 세 명의 小黑가 각각 확성기/다리 놓기/문 열기를 맡는 구도
- 小黑가 문구 도구함에 도장을 찍는 구도
- 小黑가 경고판을 들고 흔한 함정 경로를 보는 구도

같은 주제라도 새 은유로 바꾼다. 예를 들어 “이어받는 경로”는 꼭 길을 그릴 필요가 없고, 小黑가 콘텐츠 꼬리를 문손잡이에 묶는 장면으로 그릴 수 있다. “하나의 재료를 여러 용도로 쓰기”도 꼭 물고기를 그릴 필요 없이 小黑가 종이 뭉치를 여러 모양으로 눌러내는 장면으로 그릴 수 있다.
